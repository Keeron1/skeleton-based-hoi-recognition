# Café Skeleton-based Real-time HOI Recogntion

Pipeline for Human-Object Interaction recognition using:
- YOLOv8 (detection)
- DeepSORT (tracking)
- HRNet (pose estimation)
- ST-GCN (action recognition)

Run setup_main.ps1 to install the official local environment
./setup_main.ps1

If running on a setup without a GPU, run setup_cpu.ps1 to install the local environment
./setup_cpu.ps1

Run to activate local environment
.\venv\Scripts\activate
